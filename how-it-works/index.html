How Seshy Works

Seshy is a Desktop application for MacOS & Windows

Seshy is primarily a desktop application. For now you'll need to use the web site to create new teams and to invite people to your teams. But for working with project and collaborating with other people you'll primarily use the app.

Seshy supports the following DAWs:

  • Logic Pro X
  • Ableton Live
  • Avid Pro Tools
  • Bitwig Studio

The desktop app operates on your ~/Seshy folder
